# Spring-Security-JWT

## Description
This project demonstrates the implementation of JSON Web Tokens (JWT) for securing a Spring Boot application. It includes features like user authentication, authorization, and token validation.

## Features
- User authentication with JWT
- Role-based access control
- Token generation and validation
- Secure REST APIs

## Prerequisites
- Java 17 or higher
- Maven 3.8+ or Gradle
- Spring Boot 3.x
